Introduction

This web application implements a Machine Learning Framework proposed by F. Faria [1].
Given a visual classification problem, a set of descriptors and a set of learning methods, the framework will learn patterns from available instances for training in order to classify new and unseen instances.

Get started

Dataset

As an input for the application, a dataset shall be provided in .arff which is a Weka compatible format.

See example »

Configuration

Along with the Dataset, a list of classifiers and its parameters shall be provided.

Output

A report, which informs the classification´s statistics, is provided as output.
